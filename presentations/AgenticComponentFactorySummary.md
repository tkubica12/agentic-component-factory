Agentic Component Factory builds a catalog of agentic components—systems that generate, build, deploy, and validate software, then return a running interface for a main coding agent. Instead of relying on templates that still require manual edits, provisioning, seeding, deployment, and verification, it exposes MCP tools that automate the full workflow and return integration details. The first component is an API Mock Generator: given a resource name and sample JSON, it seeds data (optionally synthesizing additional records) and deploys a live FastAPI CRUD API backed by Azure Cosmos DB. A FastMCP server stores job state in Cosmos DB and queues work on Service Bus; a worker uses the GitHub Copilot SDK (Azure OpenAI BYOK) to generate code, build in ACR, deploy to Azure Container Apps, optionally generate data, and run smoke tests.
